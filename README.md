## Jekyll Theme
* [jekyll-theme-yat](https://github.com/jeffreytse/jekyll-theme-yat)

## Build
* Ruby💎 설치
  * https://rubyinstaller.org/downloads/
* jekyll🎭, bundler package 설치
  * ``` gem install jekyll bundler ```
* bundle 설치
  * ``` bundle install ```
* Server
  * ``` bundle exec jekyll serve ```
* http://127.0.0.1:4000/ 접속🙋‍♀️
